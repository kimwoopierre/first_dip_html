CSS로 웹페이지 꾸미는 법

head태그에 들어가는 style 태그는, body 에 있는 태그의 스타일을 변경해주는 역할을 한다.
strong {color:rgb} 처럼 변경하고자하는 태그의 이름에 { }을 넣고 { }안에 효과를 부여해준다.
위 예시에서, strong{ }은 style 태그를 통해 선택된 태그인 선택자(selector)라고 한다. 그리고 { }안에서 선택자에게 지정될 효과를 효과/선언(declaration) 이라고 한다.
html의 언어에서는 font태그를 통해 바꿀 수도 있지만, style태그로는 원하고자 하는 태그를 한번에 처리해줄 수 있고, 덕분에 수정이 필요할 때 수정 소모 시간이 현저히 줄어든다.
또한 코드의 수가 많아질수록, 접속인원이 늘어날수록 그에 따른 손해가 발생하기에, 코드의 수를 줄여 최대한 가볍게 만들기 위함이기도 하다.

지역변수(해당 웹페이지에서만 작동하는 변수)을 주어서 같은 입력값을 각 페이지마다 다르게 표현해줄 수 있다.
이러한 역할을 하는 것은 style 속성이다. style 속성은 지역변수를 주고 싶은 입력값이 해당한 태그에 style="color:red" 와 같은 형태로 넣어주면 된다.
text-decoration: none 과 같은 효과는 텍스트에 붙는 장식(밑줄)이 없어지게 할 수 있고, 지역변수로써 작용하게 하려면, 속성으로써의 style="text-decoration:underline"와 같이 밑줄을 넣어주면 된다.
css에서 ;을 넣어주는 의미는 각각의 효과가 시작되고 끝나는 것을 구별해주고 나눠주는 역할을 한다.

텍스트의 크기를 더 키우거나 원하는 사이즈로 맞추고 싶을 때는, style태그에서는 font-size 효과를 사용하면 되고, style 속성에서도 같은 효과를 사용하면 된다.
마찬가지로, 텍스트의 위치를 정렬(align)하기 위해서는, style 태그 혹은 style 속성에서 text-align 효과를 사용하면 된다.

선택자에는 크게 id 선택자, class 선택자, 태그 선택자가 있다.
가장 구체적인 성격을 가진, id 선택자는 태그 안에서 id=" "로 지정해줄 수 있고, 정해진 id 값은 웹페이지 내에서 한번만 쓰기로 정해져 있다.
style태그 내에서 id 선택자를 지정하기 위해서는 id값 앞에 #을 붙여주면 된다.
id 선택자 다음으로 구체적인, class 선택자는 태그 안에서 class=" "로 지정해줄 수 있고, class안에는 여러개의 값을 넣어줄 수 있다.
style태그 내에서 class 선택자를 지정하기 위해서는 class값 앞에 .을 붙여주면 된다.
가장 포괄적인 성격을 가진, 태그 선택자는 가장 큰 범주의 선택자로, 태그 자체를 지정해주는 것이다.
style태그 내에서 태그 선택자를 지정하기 위해서는 태그값을 적어주면 된다.
위 세 개의 선택자들은 코드에 적히는 순서에 따라서 적용되기도, 안되기도 한다. 나중에 나오는 선택자 일수록, 더 큰 영향력을 가지게 되고, 구체적인 순서가 먼저 나올때마다 이후에 나올 포괄적인 선택자는 적용되지 않는다.
쉽게 말해서, id 선택자가 먼저 코드 상에서 먼저 적용된다면, 이후에 id 선택자에 영향을 받는 class 선택자나 태그 선택자는 적용되지 않는 것이다.

박스모델을 이용해서 테두리를 만들고, style태그에서 바꾸고자하는 태그 선택자를 적고, 테두리를 바꾸고 싶을 때는 border를 효과를 통해 바꿀 수 있다.
border: 5px solid red; 의 경우 5px짜리 빨간색 단선을 넣는다는 의미이다.
또한 각 태그마다 이미 지정되어 있는 몇가지 효과가 있다. 이런 효과는 따로 스타일을 정의해주지 않아도 되지만, 원할 경우, 정의를 바꿔줄 수 있다.
h1과 같은 경우, 자동으로 화면 줄바꿈이 되듯, 화면 전체를 사용하는 태그이기 때문에, block element라고 하고,
반대로 a태그와 같이 입력값에만 적용되듯이, 화면의 일부만을 사용하는 태그를, inline element라고 한다.
만약 h1태그를 inline element처럼 사용하고 싶을 경우, display:inline과 같이 display 효과를 사용하면 된다. 마찬가지로 display:none은 화면에서 보이지 않게 할 수 있다.
만약 같은 스타일의 태그가 있는 경우, h1, a { }처럼, 태그 사이에 , 을 넣어서 동일한 스타일로 적용해줄 수 있다.
padding:20px의 경우, 입력값 주위의 간격을 주는 효과이고,
margin:20px의 경우, border 즉 테두리 주위의 간격을 주는 효과이다.
width:20px나 height:20px의 경우, border의 크기를 정의해줄 수 있다.

박스모델을 통해, 제목과 본문을 구별하는 선을 나눌 수 있다.
border 효과에서 border-top, border-bottom 등 원하는 쪽의 선을 지정한 뒤, 원하는 스타일의 효과를 추가해주면 된다.
이후, 웹페이지의 개발자모드를 통해, 이미 지정된 값이나 변경하고 싶은 부분의 margin값이나 padding 값을 확인하고 수정하면 된다.
웹페이지에는 margin이 붙어있기때문에 웹페이지 끝에 붙여서 출력하고자 할때는 body 태그에 margin을 0으로 설정하면 된다.

디자인 용도로만 사용되고 아무런 의미도 가지지 않는 태그에는 div태그와 span태그가 있다.
div 태그는 block element 태그이고, span태그는 inline element 태그이다.
div 태그에 id선택자를 부여하고, 그 아래에 다른 2개의 div태그들을 넣었을때, id 선택자에서 grid-template-columns: 150px 1fr; 을 주면,
id 선택자 아래에 있는 첫번째 div 태그의 가로 크기가 150px로 고정되고, 두번째 div 태그의 가로 크기가 전체화면에서 150px을 빼고 남은 부분을 차지한다.
fr은 웹페이지 창이 변하면 변하는대로 크기가 변하고, px은 설정한 값 그대로 고정된다.
또한 fr로 배정된 div 태그가 여러개 일 경우, fr 총 합에 비례해서 크기가 정해진다.
cloumns(가로)로 크기가 정해지면, 입력값이 많아지더라도 가로 비율은 변하지 않고 세로로 동일하게 늘어나게 된다.
이 사이트를 통해 사용하고자 하는 태그가 어떤 브라우저에서 사용가능하고 불가능한지 확인할 수 있다.

div 태그 아래 자손 태그들을 둘 수 있고, #id선택자(띄어쓰기)자손태그 와 같이 style 태그에서 각 div태그마다 달려있는 같은 이름의 태그 선택자들을 div내 id선택자를 통해 구분지어줄 수 있다.
예) #css_page_code strong{ color:black;}
css_page_code라는 id선택자를 가진 div태그 안에 있는 strong 태그들은 검은색이다 라는 것이다.

같은 웹페이지를 다른 화면(스마트폰 등)에 맞춰 최적화된 화면을 보여줘야 할 필요가 있다.
이를 위해 미디어쿼리라는 기능을 이용하면 된다. 웹페이지 화면에 따라 보여지는 페이지를 달리 보여주는 기능인 미디어쿼리는 @media를 이용해서 사용하면 된다.
예) @media(min-width:800px) {div{display:none}}
가로 크기가 800px보다 커질때 div태그 내의 입력값을 화면에 출력하지 않는다는 의미이다.
반대로 max-width:800px일때는 가로 크기가 800px보다 작을 때, 작동한다는 의미이다.

css로 만들어진 디자인된 페이지를 다른 페이지에 적용하게되면 디자인이 들어갈 모든 코드를 수정해야 한다.
그래서 css로 적용되는 코드들은 .css로 따로 빼서 만들어준다.
.css로 따로 뺀 코드들을 .html 파일로 적용하기 위해서는, link 태그를 이용하여 적용받을 .html 파일마다 넣어주면 된다.
예) link rel="stylesheet" href="css 파일 위치"
덕분에 동일하게 디자인이 적용되는 코드들은 .css에서 바로 적용받을 수 있고, 수정 시에도 .css에서 바로 수정 및 적용이 된다.
웹페이지를 불러올 때, 우리는 페이지에 관한 내용인 .html을 다운받게 되고, 마찬가지로 .css파일도 다운 받게 되는데, 이것을 캐시파일이라고 한다.
이 캐시파일은 한번 다운되면, 캐시파일 내 .css파일이 변경되지 않는 한, 다시 다운 받지 않아도 된다.

새로운 줄을 추가할때는 br 태그를 사용한다. br태그는 닫히는 태그 없이 사용 가능하고, 한번에 여러개 사용해서 단락으로써 여백을 줄 수 있다.
또한 p라는 태그를 이용해서 단락을 설정해줄 수 있다. p는 열리는 태그와 닫히는 태그가 존재하고, p태그로 감싼 입력값은 하나의 단락으로 정의된다.
이렇게 정의된 단락이 코드 상의 정보를 주기 때문에 더 좋은 태그라고 할 수 있고, p태그를 사용하는 것은 권장한다.

br과 p 태그를 비교했을 때, br은 각 문장 뒤에 쉽게 쓸 수 있고 여러번 사용해서 여백을 더 줄 수도 있지만, p 태그는 전체문장을 하나의 단락으로 묶는 것이기에 자유도가 떨어진다.
그래서 CSS 언어에서 그것을 보완할 수 있다. CSS에는 p와 p 사이에 여백을 주는 코드가 있는데, 새로운 p가 시작되는 태그 안에, style=을 적고, "margin-top:XXpx;"이라는 XX 픽셀만큼의 top 부분에 여백을 주는 코드를 넣어 주면 된다.

title이란 태그를 통해 웹페이지 제목을 정해줄 수 있다. 이 제목은 사람들로 하여금 웹페이지의 표지의 제목과 같은 것이라고 할 수 있다.
또한 meta라는 태그를 이용해 웹페이지의 성격을 정의해줄 수 있다. meta 태그 안에 charset=""을 적고, ""안에 utf-8을 넣어주면 한글을 모든 웹사이트에서 오류 없이 읽을 수 있게 해준다.
위 title 태그와 meta 태그는 페이지에 대한 설명에 관한 태그로, head라는 태그로 감싸고, 그 아래 페이지 내 내용에 대한 태그들은 body라는 태그로 감싸준다.
이 head태그와 body태그를 하나로 묶어 html이라는 태그로 감싸주고, 제일 상위에 관용적으로 !doctype html이라는 태그로 정의해준다.

a 태그에 href=""을 적고 ""안에 링크 주소를 넣어 a태그로 묶인 입력값을 눌렀을 때 링크 주소로 이동할 수 있고, target="_blank"를 넣어 새 탭을 열어 링크 주소로 이동할 수도 있다. 또한 title=""을 넣어 ""안의 내용을 입력값 위에 커서를 놓았을 때, ""안의 내용을 출력한다.
